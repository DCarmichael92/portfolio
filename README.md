# Devin Carmichael Portfolio

This repository contains my personal portfolio website built with Next.js and Tailwind CSS. The site showcases my software engineering experience, featured projects, work background, and personal interests.

## Features

- Modern web stack using Next.js and React
- Tailwind CSS for custom styling
- Fully responsive design
- Travel visualizations and project sections
- Deployed on Vercel and accessible at:
  https://devincarmichael.com

## Tech Stack

- Next.js 16
- React
- TypeScript
- Tailwind CSS
- Vercel

## Structure

- `src/app` - Next.js app routes and components
- `src/components` - Reusable UI components
- `src/data` - Data for projects, experience, and travel
- `public` - Static assets


## Deployment

The site is deployed via Vercel. Any push to the main branch triggers a new deployment.
